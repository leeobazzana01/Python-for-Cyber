# Python para Segurança

Bem-vindo ao **Python para Segurança**, um repositório dedicado ao aprendizado prático e teórico de **Python focado em segurança**, organizado do básico ao avançado.

---

## Objetivo

- Aprender **Python para automação e segurança**: manipulação de arquivos, redes, criptografia, fuzzing e exploits.  
- Criar scripts e ferramentas práticas para **pentest, análise e automação de tarefas de segurança**.

---

## Pré-requisitos

- **Python 3.x**  
- **Editor de código** (VS Code, PyCharm, etc.)  
- **Noções básicas de Linux e terminal**

---

## Estrutura do Repositório

- **Python/**  
  - Básico → Sintaxe, tipos, listas, dicionários, controle de fluxo, funções  
  - Intermediário I → Arquivos, tratamento de erros, regex, CLI  
  - Intermediário II → Redes, protocolos, sockets, HTTP, cliente-servidor  
  - Avançado I → Hashes, criptografia, OOP, threads e multiprocessing  
  - Avançado II → Web scraping, pacotes, fuzzing, integração com APIs  
  - Expert → Desenvolvimento de exploits, payloads, evasão e automação avançada

---

## Como Usar

1. Escolha o nível que deseja estudar.  
2. Abra os exemplos no seu editor de código.  
3. Execute os scripts Python.  
4. Crie projetos baseados nos exemplos para praticar.
